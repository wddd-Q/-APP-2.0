## 一、通用模式

### 三态按钮

| 状态 | 视觉 | 示例 |
|------|------|------|
| 可用 | 正常色 | [建造 200灵石] |
| 不可用 | 灰色 + 提示 | [建造 灵石不足] |
| 进行中 | 旋转图标 | [建造中... 剩余2月] |

### 数值显示

```
灵石显示: [灵石图标] 1,200
突破概率: 45% (绿: >50%, 黄: 20-50%, 红: <20%)
进度条: ████████░░ 80%
冷却中: 灰色 + [剩余3月]
```

### 弹窗模式

```
┌──────────────────────┐
│ 散修来投              │ ← 标题
│                      │
│ 一位筑基期散修慕名而   │ ← 描述
│ 来，希望加入宗门...    │
│                      │
│ [收留]  [拒绝]  [考验]│ ← 选项（2-4个）
└──────────────────────┘
```

## 二、领域模式

### 弟子列表

```
每行显示:
[姓名] | [境界·小阶段] | [灵根] | [当前任务▼] | [年龄] | [详情→]

排序: 境界降序（掌门/长老优先）
筛选: 按境界/任务/灵根过滤
搜索: 姓名搜索
```

### 设施管理

```
设施卡片:
┌─────────────────┐
│ 修炼室 Lv.2      │
│ ─────────────── │
│ 容量: 3/6人     │
│ 修炼加成: +40%  │
│ 维护: 20灵石/月 │
│                 │
│ [升级 500灵石]  │
└─────────────────┘
```

### 时间推进反馈

```
推进一月后, 顶部通知区:
✓ 张三 突破至 筑基后期
✓ 灵脉产出 +10灵石
⚠ 灵石月结余为负（-15/月）
```

---

## 三、反馈时机

| 操作 | 即时反馈 | 延迟反馈 |
|------|----------|----------|
| 建造设施 | 按钮变灰 + 建造中 | 建造完成通知 |
| 招收弟子 | 弟子出现在列表中 | — |
| 分配任务 | 任务标签更新 | 月度产出 |
| 推进时间 | 顶栏数字刷新 | 突破/事件弹窗 |
| 炼丹 | 进度条 + 等待 | 丹药成品弹窗 |

---

## 四、错误与边界

| 场景 | 处理 |
|------|------|
| 灵石不足 | 按钮禁用 + 红色文字说明缺多少 |
| 弟子上限已满 | [招收] 按钮禁用 |
| 设施槽位已满 | [建造] 按钮不显示 |
| 突破失败 | 保留进度条 + 鼓励性提示"再接再厉" |
| 弟子死亡 | 名册中变灰 + "坐化于宗门历XX年" |
| 存档失败 | 弹窗提示 + 建议检查磁盘空间 |
