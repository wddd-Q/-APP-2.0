# 玩家旅程 — 修仙：宗门风云

## 元数据

| 字段 | 内容 |
|------|------|
| **游戏** | 修仙：宗门风云 |
| **日期** | 2026-05-27 |
| **状态** | 草案 |

---

## 一、首次接触（0-5 分钟）

### 到来情绪
好奇、期待。"这个修仙经营游戏长什么样？"

### 玩家核心问题
- 这是什么游戏？
- "我要做什么？"
- "我能建立自己的宗门吗？"

### 必须交付的关键体验
1. **开场动画/静态画面**: 一座衰败的山门，几位老弱弟子，一片荒芜的灵田
2. **第一印象**: 水墨风美学，古琴背景音乐，界面干净不拥挤
3. **初始状态**: 自动创建一个九品小宗门，掌门（玩家名），3-5个初始弟子（属性随机但保证有一个可塑之才）
4. **第一个交互**: 弹出"招收新弟子"引导，引导玩家发布招徒令

### 退出情绪
"这个开局有意思，我想看看这3个弟子将来能成什么。"

### 失败风险
- 信息过载 — 一上来展示太多面板和数字 → 使用渐进式引导
- 决策焦虑 — 不知道该先做什么 → 首月有任务引导系统

---

## 二、方向指引（5-30 分钟）

### 到来情绪
初步适应，开始探索。"这东西怎么玩？有什么可以做的？"

### 玩家核心问题
- "怎么让弟子变强？"
- "灵石从哪来？"
- "怎么扩张宗门？"

### 必须交付的关键体验
1. **第一个修炼循环**: 指引玩家将一个弟子放入修炼室 → 推进一个月 → 看到修炼进度增加
2. **第一次资源获取**: 发现宗门后山有低品灵脉，开始产出下品灵石
3. **第一座建筑**: 建造第二座修炼室或丹药堂，看到宗门布局变化
4. **突破初体验**: 一个月后，最好的那个弟子突破到练气中期

### 必须建立的认知
- 时间推进（月/年）是关键机制，"推进时间"按钮是核心操作
- 弟子任务分配是核心玩法
- 灵石是一切的基础
- 宗门界面 = 主界面

### 退出情绪
"我明白了，这就是个经营宗门培养弟子的游戏，每个月要做资源决策。"

---

## 三、首次掌握（30 分钟 — 2 小时）

### 到来情绪
有一定理解，开始自主规划。"我想让李青云突破到筑基期。"

### 玩家核心问题
- "怎么帮弟子突破筑基？"
- "炼丹是怎么回事？"
- "灵根重要吗？"

### 必须交付的关键体验
1. **第一次大突破**: 一个练气九层弟子 + 筑基丹（玩家炼丹/坊市购买）→ 突破筑基 → 寿元延长 → 玩家理解突破的重要性
2. **炼丹入门**: 收集灵草 → 丹方 → 炼制第一炉丹药（可能失败）→ 理解炼丹系统
3. **灵根发现**: 招收到一个天灵根弟子 → 明显感受到修炼速度差异 → 理解灵根价值
4. **第一次随机事件**: 天降陨石（内含稀有材料）/ 散修拜访 / 妖兽袭击边缘村庄
5. **资源决策点**: 灵石有限，是先升级修炼室还是先买灵草炼丹？第一次有意义的资源博弈

### 退出情绪
"筑基丹真难炼啊但炼出来了！下次我要让更多弟子突破。"

---

## 四、深度发现（2-10 小时）

### 到来情绪
熟悉基础系统，开始探索深度。"还有什么隐藏机制？"

### 玩家核心问题
- "金丹期需要什么条件？"
- "怎么和其他宗门打交道？"
- "弟子之间会有什么关系？"
- "什么是天劫？"

### 必须交付的关键体验
1. **金丹天劫**: 第一个弟子突破金丹 → 天劫降临 → 成功/失败的高悬念时刻
2. **宗门外交**: 第一次收到其他宗门的来访/同盟请求/挑衅 — 关系系统开启
3. **弟子关系**: 发现两个弟子"结为道侣"的事件 → 关系网系统自然展现
4. **世界事件**: 正魔大战爆发 / 秘境出世 — 世界不仅有你
5. **第一代传承**: 一个初始老弟子寿元将尽 → 坐化/突破 → 代际更替的感动
6. **宗门升级**: 从九品升至八品 → 设施上限+1 → 可招收更多弟子

### 退出情绪
"这个世界比我想象的大，其他宗门的动向也需要关注了。"

---

## 五、习惯性游玩（10-50 小时）

### 到来情绪
深度投入，形成自己的策略。

### 玩家核心问题
- "我的宗门风格是什么？正道/魔道/中立？"
- "怎么培养一个元婴强者？"
- "灵脉战争值不值得打？"

### 必须交付的关键体验
1. **门派特色确立**: 玩家选择专精方向 — 炼丹大派 / 剑修宗门 / 阵法世家
2. **元婴强者诞生**: 第一个元婴修士出现 → 宗门在区域内名声大噪
3. **灵脉战争**: 与邻宗争夺一条上品灵脉 → 第一次宗门规模的战斗
4. **师徒传承**: 第二代弟子（元婴修士收的徒弟）开始崭露头角
5. **宗门排名**: 地区宗门排名系统中持续攀升

### 留存机制
| 时刻 | 钩子 |
|------|------|
| 回合开始 | 查看昨晚离线期间发生了什么 |
| 突破前夕 | "弟子张三还有3个月就能突破金丹了！" |
| 炼丹等待 | "还差最后一味材料…派弟子出去找" |
| 代际节点 | "老掌门是否该退位传给弟子？" |
| 世界大事件 | "正魔大战要来了，该选哪边？" |

### 退出情绪
"我的宗门已经八品了，明天争取升七品。"

---

## 六、长期参与（50+ 小时）

### 到来情绪
资深玩家心态。"让我试试不同的玩法。"

### 玩家核心问题
- "飞升的终极目标是什么？"
- "魔道路线有什么不同？"
- "最小宗门/最大宗门挑战？"

### 必须交付的关键体验
1. **飞升终点**: 宗门培养出飞升修士 → 飞升劫 → 成为传说中的宗门
2. **多周目差异**: 正道路线 vs 魔道路线 → 不同事件池、不同外交关系
3. **自定义挑战**: 单灵根宗门 / 废灵根逆袭 / 不主动战斗的和平路线
4. **宗门传奇**: 回看宗门历史 — 历任掌门、传奇弟子、重大事件年表

## 反模式清单

- [ ] 教程超过10分钟 → 分阶段引导，不要一次教完
- [ ] 首次失败没有恢复机制 → 第一次突破失败不要惩罚太重
- [ ] 隐藏核心机制 → 灵根解释、突破条件在界面有提示
- [ ] 玩家不知道下一步做什么 → 始终有一个可选的任务引导

---

## 验证问卷（内部测试用）

1. 5分钟后：你能描述这个游戏的玩法吗？
2. 30分钟后：你现在最想完成的目标是什么？
3. 2小时后：你最喜欢哪个弟子？为什么？
4. 10小时后：你觉得自己宗门和其他宗门有什么区别？
5. 如果中断一周后回来玩：你能马上知道宗门当前状态吗？
