## 一、设计哲学

"每局都是独特的故事" — 事件不按固定时间线触发，而是基于条件组合的涌现式叙事。

### 设计原则
1. **无硬编码故事线**: 没有"第10年必发生的XX战争"
2. **条件驱动**: 事件由世界状态+弟子状态+随机因子组合触发
3. **事件链**: 事件可产生后果，后果引发新事件
4. **不做完所有可能**: 留白 — 不是所有条件组合都有事件

## 三、触发机制

### 条件权重系统

每个事件有一个基础权重和触发条件：

```
事件触发概率 = (基础权重 / 总权重)
    × 条件满足系数
    × 随机因子

每回合（月）从满足条件的事件池中随机抽取 0-3 个事件
```

### 触发条件类型

| 条件类型 | 示例 |
|----------|------|
| 时间条件 | month ∈ [3,5]（春季）、year % 100 == 0（百年） |
| 宗门状态 | rank ≤ 7、prestige > 500、karma < -100 |
| 弟子状态 | realm ≥ 3、mentality < 30、fortune > 80 |
| 世界状态 | era ≥ 2、正魔大战进行中 |
| 历史条件 | 未触发过事件X、已触发事件Y |
| 设施条件 | 拥有丹药堂 Lv.3、拥有藏经阁 |

### 事件冷却

| 事件等级 | 冷却时间 | 说明 |
|----------|----------|------|
| 普通 | 3月 | 弟子级琐事 |
| 不凡 | 6月 | 宗门级事件 |
| 稀有 | 1年 | 区域级事件 |
| 史诗 | 3年 | 重要事件 |
| 传说 | 10年 | 改变世界格局 |

## 四、事件链系统

### 三幕结构

```
触发事件 → 玩家选择 → 产生后果标记 → 后续事件可检测标记
```

### 事件链示例: "散修来投"

```
第一幕: 一名散修前来投靠宗门
  选择A: 收留（消耗灵石，获得弟子）
  选择B: 拒绝（无消耗，声望轻微下降）
  选择C: 考验后再决定（可能获得更强弟子，也可能散修恼怒离去）

第二幕（若选A）: 该弟子展现出隐藏天赋
  → 触发"灵根觉醒"事件（该弟子灵根提升一级）

第三幕（若灵根觉醒）: 该弟子的仇家找上门
  → 触发"仇家寻仇"事件（宗门战或外交谈判）
```

## 五、事件池设计

### MVP 初始事件池（16个）

**宗门级（6个）**
1. 散修来投 — 招收/拒绝/考验
2. 妖兽袭击 — 战斗/谈判/驱赶
3. 商队路过 — 交易/劫掠/保护
4. 灵脉异动 — 投入资源探索/忽略
5. 天降陨石 — 采集材料/危险规避
6. 弟子发现秘洞 — 派遣探索/封存

**弟子级（5个）**
7. 修炼感悟 — 修为大增/普通收获
8. 心魔发作 — 需丹药救助/自行克服/放弃
9. 弟子争执 — 调解/放任/惩罚
10. 偶得异宝 — 上交宗门/私藏/分享
11. 道心动摇 — 劝导/无视/逐出师门

**世界级（5个）**
12. 正魔大战 — 参战/中立/渔翁得利
13. 秘境出世 — 派遣弟子/亲自前往/无视
14. 异宝出世 — 争夺/放弃
15. 上古遗迹 — 探索/保守观望
16. 百年灵气潮汐 — 全体修炼加速3年

## 六、叙事积木

### 弟子人格标签

弟子生成时随机分配2-3个人格标签，影响事件中的行为和选项：

| 标签 | 影响 |
|------|------|
| 勇猛 | 战斗事件中+成功率 |
| 谨慎 | 探险事件中-风险 +安全 |
| 贪婪 | 可能私藏资源 |
| 忠诚 | 不会被敌对宗门收买 |
| 孤傲 | 社交事件-好感 |
| 好奇 | 探索事件+发现率 |
| 善良 | 正面选项权重增加 |
| 阴狠 | 可能毒害同门 |

### 叙事模板

```
"[弟子A]因[原因]，在[地点][做了什么]，导致[结果]"

原因池: {人格标签} + {当前状态} + {随机}
地点池: 修炼室/后山/坊市/秘境/外出的路上
结果池: {获得物品} / {属性变化} / {关系变化} / {触发新事件}
```

## 七、世界大事件

### 纪元系统

| 纪元 | 持续时间 | 效果 | 触发条件 |
|------|----------|------|----------|
| 和平纪元 | 100-300年 | 贸易繁荣、修炼资源充足 | 默认 |
| 乱世纪元 | 50-100年 | 战斗频发、灵石价格飞涨 | 世界大战触发 |
| 仙缘纪元 | 30年 | 天灵根弟子出现率×5 | 灵气潮汐触发 |
| 末法纪元 | 100年 | 灵气稀薄、修炼速度减半 | 世界线事件 |
