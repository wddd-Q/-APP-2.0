## 一、设计原则

这是一个**门派模拟经营游戏**，不是动作RPG。战斗是战略层面的事，不由玩家手操。外交系统参考《三国志》《十字军之王》的势力间交互模式。

## 二、宗门声望

### 声望等级

| 等级 | 声望值 | 效果 |
|------|--------|------|
| 默默无闻 | 0-100 | 招收弟子困难（候选人-2） |
| 小有名气 | 101-500 | 正常招收 |
| 声名远播 | 501-2000 | 招收候选人+2，吸引优质弟子 |
| 如雷贯耳 | 2001-10000 | 天灵根弟子出现率×2 |
| 名震天下 | 10001-50000 | 其他宗门主动示好 |
| 传说 | 50000+ | 飞升修士回馈宗门概率提升 |

| 行为 | 声望变化 |
|------|----------|
| 弟子突破金丹+ | +50-100/次 |
| 宗门等级提升 | +200-500 |
| 赢得灵脉战争 | +300 |
| 弟子天劫成功 | +100 |

## 三、阵营系统

### 三大阵营

| 阵营 | 特征 | 机制差异 |
|------|------|----------|
| **正道** | 维护秩序、禁止魔功 | 正派宗门外交友好、丹方交易优惠 |
| **魔道** | 不择手段、追求力量 | 魔功修炼速度快但风险高、可能被正道围攻 |
| **中立** | 明哲保身、见机行事 | 灵活外交、但两边都不完全信任 |

### 业力值（Karma）

```
业力范围: -100（极恶）← 0（中立）→ +100（大善）

正向行为: 保护凡人、帮助弱小宗门、拒绝魔道诱惑 → +Karma
负向行为: 屠杀、劫掠、修魔功、背信弃义 → -Karma

Karma影响:
- >50: 被正道宗门视为盟友
- <-50: 被正道宗门视为敌人
- <-80: 可能触发"正道联盟围剿"事件
```

## 四、外交关系

### 关系等级

| 等级 | 数值范围 | 行为 |
|------|----------|------|
| 死敌 | -100 到 -61 | 随时可能开战 |
| 敌对 | -60 到 -21 | 贸易禁运、言语挑衅 |
| 冷淡 | -20 到 20 | 无特殊互动 |
| 友好 | 21 到 60 | 贸易优惠、情报共享 |
| 同盟 | 61 到 100 | 共同防御、联姻可能 |

### 外交行为

| 行为 | 效果 | 消耗 |
|------|------|------|
| 派遣使节 | 关系+10-20 | 灵石200 |
| 赠送礼物 | 关系+（礼物价值/50） | 物品 |
| 贸易协定 | 关系+5，双方灵石收入+10% | 外交技能 |

| 背弃盟约 | 同盟破裂，声望-1000 | — |
| 合并宗门 | 吸收弱小宗门 | 需敌对+战胜 |

## 五、战争系统

### 宣战条件

- 死敌关系自动概率性触发战争
- 灵脉争夺（有相邻领土 + 灵脉资源冲突）
- 玩家主动宣战（需声望不为负）

### 战争流程

```
1. 宣战 → 2. 调集兵力 → 3. 生成战斗报告 → 4. 战后结算
```

### 兵力计算

```
宗门战力 = Σ(弟子战斗力)

弟子战斗力 = 境界系数 × 根骨/50 × 战斗技能 × 装备加成 × 功法加成

境界系数:
  练气: 10
  筑基: 50
  金丹: 200
  元婴: 800
  化神: 3000
  炼虚: 12000
  合体: 50000
  大乘: 200000
```

### 战斗结果

```
战力比 = 进攻方战力 / 防守方战力

战力比 > 2.0: 压倒性胜利（损失<10%）
战力比 1.5-2.0: 优势胜利（损失10-25%）
战力比 1.0-1.5: 微弱胜利（损失25-50%）
战力比 0.7-1.0: 惨胜（损失50-75%）
战力比 < 0.7: 战败（损失75%+，高概率弟子阵亡）

防守方有额外加成: ×1.3（熟悉地形+护山大阵）
```

### 战后结算

**胜利方:**
- 获得赔款/领土
- 声望提升
- 活的弟子获得战斗经验

**战败方:**
- 支付赔款/割让灵脉
- 声望下降
- 部分弟子可能阵亡/被俘
- 设施可能损坏

### 弟子阵亡判定

```
阵亡概率 = 基础阵亡率 / (根骨/50) / (境界系数/100)

基础阵亡率:
  战力比 > 2.0: 1%
  战力比 1.5-2.0: 5%
  战力比 1.0-1.5: 10%
  战力比 0.7-1.0: 20%
  战力比 < 0.7: 35%
```

## 六、灵脉争夺

### 灵脉地图

世界地图上有有限数量的灵脉节点：
- 下品灵脉: 20个（产出10-50灵石/月）
- 中品灵脉: 8个（产出5-20中品灵石/月）
- 上品灵脉: 3个（产出3-10上品灵石/月）
- 极品灵脉: 1个（产出1-5极品灵石/月）

### 争夺规则

1. 无主灵脉: 先到先占
2. 有主灵脉: 需通过战争或外交获取
3. 灵脉枯竭: 开采过度可能触发（随机事件）

## 七、NPC 宗门 AI（自主运转核心）

> 此系统是 P4（修真世界）的关键实现。NPC宗门必须拥有独立的修炼、扩张、外交行为闭环，否则世界只是"等待玩家触发的剧本"。

### 自主行为闭环

```
NPC宗门年度循环:
  1. 修炼推进 → 所有NPC宗门弟子获得修为（与玩家同步推进）
  2. 资源结算 → 灵脉产出、设施维护、弟子俸禄
  3. 突破检测 → 检查是否有弟子满足突破条件 → 执行突破（含天劫）
  4. 外交决策 → 根据个性参数+当前状态选择一个目标
  5. 行动执行 → 宣战/同盟/贸易/扩张/灵脉争夺
  6. 状态广播 → 产生世界消息（玩家可见或隐藏）
```

### 行为目标（优先级排序）

1. **生存**：战力不足、面临死敌威胁时 → 寻求同盟、收缩防御
2. **扩张**：战力充沛、有可争夺灵脉时 → 发起灵脉战争
3. **发展**：和平时期 → 修炼、升级设施、提升宗门等级
4. **外交**：需要盟友时 → 结盟、联姻、贸易

### 个性参数

| 参数 | 范围 | 影响 |
|------|------|------|
| 侵略性 | 0-100 | 宣战倾向（>70 主动寻战） |
| 外交倾向 | 0-100 | 同盟意愿（>60 寻求结盟） |
| 发展优先 | 0-100 | 修炼 vs 扩张平衡（>50 重修炼） |
| 守约度 | 0-100 | 遵守盟约概率（<30 可能背刺） |

### 初始 NPC 宗门（MVP 数量: 10-15个）

每个宗门生成时随机：宗主境界（金丹/元婴）、宗门等级（5-8品）、阵营（正/魔/中立）、个性参数、初始灵脉数量。

### 年度决策算法

```
function npc_annual_decision(sect):
    threat_level = assess_threats(sect)       # 评估当前威胁
    expansion_opportunity = scan_spirit_veins(sect)  # 可扩张目标

    if threat_level > sect.combat_power * 0.7:
        return DIPLOMACY  # 寻求同盟
    elif expansion_opportunity and sect.aggression > 50:
        return WAR  # 发起灵脉战争
    elif sect.resources > expansion_threshold:
        return EXPAND  # 升级设施/招收弟子
    else:
        return CULTIVATE  # 专注修炼

    # 所有决策结果记录到世界状态，玩家通过事件/消息获知
```

### 玩家可见性

- 已知宗门的重大变化（升级/战争/有弟子突破）→ 产生世界消息
- 未知宗门的内部事务 → 对玩家隐藏
- 可通过"情报"任务获取隐藏信息

### 世界平衡

- NPC宗门经济遵循相同的经济模型（见 economy-model.md）
- NPC宗门突破使用相同的修炼公式（见 cultivation-system.md）
- 防止单一NPC宗门无限膨胀：维护成本随规模指数增长

## 八、战斗简化原则

> **这是模拟经营游戏，不是战术战棋。**

- 战斗100%自动结算，玩家看报告
- 玩家能做的决策在战前：派哪些弟子、是否使用阵法、是否请同盟
- 战斗中玩家唯一互动：是否撤退（看损失到一定程度）
- 战斗结果是概率驱动的，但所有概率对玩家可见
