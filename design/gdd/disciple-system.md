## 一、弟子属性系统

### 六维属性（0-100）

| 属性 | 作用 | 影响力 |
|------|------|--------|
| **根骨** | 突破成功率加成、天劫存活率 | ⭐⭐⭐⭐⭐ |
| **悟性** | 领悟功法速度、技能学习速度 | ⭐⭐⭐⭐ |
| **福缘** | 奇遇触发率、秘境发现、资源采集 | ⭐⭐⭐ |
| **心性** | 抵抗心魔、渡心魔劫、忠诚度 | ⭐⭐⭐⭐ |
| **魅力** | 招收弟子效果、外交加成、弟子关系 | ⭐⭐ |
| **资质** | 修炼速度（修为获取倍率） | ⭐⭐⭐⭐⭐ |

### 初始属性生成

```
新弟子总属性点 = 200 + d60（随机，200-260之间分布）
单个属性上限 = 80
单属性下限 = 10

灵根品质影响：
- 天灵根：总属性+40
- 异灵根：总属性+20
- 废灵根：总属性-20
```

### 属性提升途径

| 途径 | 效果 | 限制 |
|------|------|------|
| 修炼 | 资质+1/年（缓慢自然增长） | 上限80 |
| 丹药 | 永久+属性/丹药 | 每种丹药限用3次 |
| 功法 | 修炼特定功法获得属性加成 | 受悟性限制 |
| 奇遇 | 随机属性大幅提升 | 稀有 |
| 天材地宝 | 稀有材料直接提升 | 极稀有 |

## 二、灵根与属性判定

### 招收弟子流程

1. **发布招徒令**: 消耗灵石，等待1-3个月
2. **候选人池**: 系统生成2-5个候选人（姓名/性别/年龄/属性/灵根）
3. **灵根测试**: 可选消耗灵石进行灵根检测（否则灵根显示为"未知"）
4. **招收决策**: 选择是否招收（可多选）

### 每一代候选人生成算法

```
func generate_candidate():
    age = rand_range(14, 25)
    elements = _roll_elements()  # 随机五行属性
    quality = _determine_quality(elements)  # 根据属性组合判定品质

    # 属性生成（灵根越好，基础属性越高）
    base_points = 200 + randi() % 61  # 200-260
    if quality == "heaven": base_points += 40
    elif quality == "variant": base_points += 20
    elif quality == "waste": base_points -= 20

    distribute_points(base_points)  # 随机分配到6属性
```

## 三、功法系统

### 功法品阶

| 阶 | 名称 | 稀有度 | 属性加成 | 修炼难度 |
|----|------|--------|----------|----------|
| 4 | 天阶 | 极稀有 | +40 | 极高 |
| 3 | 地阶 | 稀有 | +25 | 高 |
| 2 | 玄阶 | 不常见 | +15 | 中 |
| 1 | 黄阶 | 常见 | +8 | 低 |

每阶又分为初/中/高/顶四品，对应 +0/+0.25/+0.5/+1 加成倍率。

### 功法类型

| 类型 | 主要加成属性 | 示例 |
|------|-------------|------|
| 心法 | 资质、根骨 | 《青云心法》《大衍心经》 |
| 功法 | 根骨、资质 | 《金刚不坏功》《九转金身诀》 |
| 身法 | 福缘、魅力 | 《凌波微步》《逍遥游》 |
| 秘术 | 悟性、心性 | 《搜魂术》《天机推演》 |

### 功法领悟

```
领悟成功率 = 基础概率 × (悟性/50) × 藏经阁加成
领悟时间 = 基础时间 / (悟性/50) / 藏经阁加成

品阶越高，基础概率越低，基础时间越长
```

### 弟子功法槽

每个弟子可修炼功法数量受境界限制：
- 练气: 2个
- 筑基: 3个
- 金丹: 4个
- 元婴+: 5个

| 技能 | 作用 | 提升方式 |
|------|------|----------|
| 炼丹 | 丹药成功率、品质 | 炼丹实践 |

### 技能熟练度

```
提升速度 = 悟性/50 × 设施加成 × 是否有师父传授
熟练度需求: 等级² × 10（即1→2需要10点，98→99需要970点）
```

## 五、弟子关系系统

### 关系类型

| 类型 | 效果 | 产生条件 |
|------|------|----------|
| **师徒** | 师父技能传授效率+30% | 指定一名高阶弟子为导师 |
| **道侣** | 双方修炼速度+10% | 好感度达到阈值后自然触发 |
| **挚友** | 战斗士气+15% | 长期同任务/共患难 |
| **仇敌** | 有概率触发门内冲突事件 | 好感度为负+事件触发 |

### 关系值（-100 到 100）

```
每月自然衰减: ±1（趋向0）
共同修炼: +2/月
共同战斗: +5/次
争夺资源: -10/次（事件触发）
境界差异过大: -1/月（超过2个大境界）
```

### 代际传承

- 弟子可以收徒（境界≥金丹，且宗门有空余师徒名额）
- 师徒关系绑定后不可轻易解除
- 师父坐化后，徒弟获得"传承"buff（属性+5%）

### 寿元管理

| 境界 | 寿元上限 | 衰老开始 |
|------|----------|----------|
| 练气 | 120 | 80岁 |
| 筑基 | 250 | 180岁 |
| 金丹 | 500 | 350岁 |

## 七、任务分配系统

### 可分配任务

| 任务 | 产出 | 消耗 |
|------|------|------|
| **闭关修炼** | 修为进度 | 无（占用修炼室位置） |
| **炼丹** | 丹药 | 材料 + 灵石 |
| **炼器** | 法器 | 材料 + 灵石 |
| **外出游历** | 随机（材料/情报/奇遇） | 可能受伤/陨落 |
| **镇守山门** | 安全度 | 无 |
| **教导弟子** | 徒弟技能 | 自身修炼减速20% |
| **坊市交易** | 买低卖高 | 时间 |
| **采集灵草** | 灵草 | 灵田产出 |
| **采矿** | 矿石 | 矿脉产出 |

### 任务效率

```
任务效率 = 对应属性/50 × 对应技能/50 × 设施加成
```

## 八、弟子 AI 行为

弟子在无分配任务时，会自主行为：
- 优先修炼（如果修炼室有空位）
- 偶尔与他人社交（关系变化）
- 低概率触发个人事件（感悟/争吵/发现）
