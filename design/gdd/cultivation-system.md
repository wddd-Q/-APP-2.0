## 概述

修炼体系是游戏最核心的纵向成长系统。境界突破是弟子成长的主轴，灵根决定了弟子的天赋上限，天劫是成长道路上的关键风险节点。

## 一、境界体系

### 九大境界

| 阶 | 境界 | 小阶段数 | 寿元(年) | 突破难度 | 天劫 | 关键需求 |
|----|------|----------|----------|----------|------|----------|
| 1 | **练气** | 9层 | 120 | DC 8→17 | 无 | 灵气感应 |
| 2 | **筑基** | 初/中/后/圆满 | 250 | DC 18→22 | 无 | 筑基丹 |
| 3 | **金丹** | 初/中/后/圆满 | 500 | DC 24→30 | 小天劫 | 结金丹 |
| 4 | **元婴** | 初/中/后/圆满 | 1000 | DC 32→38 | 四九天劫 | 结婴丹 |
| 5 | **化神** | 初/中/后/圆满 | 2000 | DC 40→46 | 六九天劫 | 化神丹 |
| 6 | **炼虚** | 初/中/后/圆满 | 4000 | DC 48→54 | 心魔劫 | 悟道（心性≥80） |
| 7 | **合体** | 初/中/后/圆满 | 8000 | DC 56→62 | 九九重劫 | 炼虚丹 |
| 8 | **大乘** | 初/中/后/圆满 | ∞ | DC 64→70 | 飞升劫 | 道基圆满 |
| 9 | **飞升** | — | ∞ | — | — | — |

### 小阶段突破

每个大境界内的小阶段突破遵循：
```
小阶段DC = 基础DC + (小阶段序号 × 坡度)
坡度 = (最大DC - 最小DC) / 小阶段数
```

例：练气期（DC 8-17，9层）→ 每层约+1 DC
例：筑基期（DC 18-22，4阶段）→ 每阶段约+1 DC

### 大境界突破

跨越一个大境界时有特殊机制：
- **瓶颈**: 大境界最后阶段突破失败时，进度不归零但减少30%
- **药引**: 需要对应丹药（无丹药则DC翻倍）
- **天劫**: 金丹开始每次大突破都有天劫

## 二、突破机制

### 突破公式

```
实际成功率 = CLAMP(
    基础概率(灵根修正)
    + 丹药加成
    + 环境加成(设施等级)
    - 心魔惩罚
    - 突破次数惩罚
, 5%, 95%)
```

### 参数详解

#### 基础概率
| 当前阶段 | 基础成功率 |
|----------|-----------|
| 练气各层 | 80% → 30%（随层递减） |
| 筑基→金丹 | 25% |
| 金丹→元婴 | 15% |
| 元婴→化神 | 8% |
| 化神→炼虚 | 4% |
| 炼虚→合体 | 2% |
| 合体→大乘 | 1% |
| 大乘→飞升 | 0.5% |

#### 灵根修正（乘数）
| 灵根品质 | 修正系数 |
|----------|----------|
| 天灵根 | ×2.0 |
| 异灵根 | ×1.5 |
| 真灵根 | ×1.0 |
| 伪灵根 | ×0.6 |
| 废灵根 | ×0.3 |

#### 丹药加成（加数）
| 丹药 | 加成(%) | 适用突破 |
|------|---------|----------|
| 筑基丹 | +20% | 练气→筑基 |
| 结金丹 | +15% | 筑基→金丹 |
| 结婴丹 | +12% | 金丹→元婴 |
| 化神丹 | +10% | 元婴→化神 |
| 炼虚丹 | +8% | 化神→炼虚 |
| 合体丹 | +5% | 炼虚→合体 |
| 大乘丹 | +3% | 合体→大乘 |

#### 环境加成
| 设施 | 加成 | 条件 |
|------|------|------|
| 修炼室 Lv.1 | +3% | 在修炼室突破 |
| 修炼室 Lv.3 | +8% | 修炼室满级 |
| 聚灵阵 | +5% | 需要阵法Lv.2 |
| 灵脉品质 | +等级% | 宗门所在灵脉 |

#### 心魔惩罚
| 条件 | 惩罚(%) |
|------|---------|
| 心性 < 30 | -10% |
| 心性 30-50 | -5% |
| 心性 > 80 | +0%（免疫） |
| 杀性 > 70 | -8%（心魔深重） |
| 有未了怨仇 | -5% |

#### 突破次数惩罚
同一境界内突破失败次数累加惩罚：
- 第1次失败: -0%
- 第2次失败: -3%
- 第3次失败: -6%
- 第4次+: -10%（封顶）

### 失败结果表

| 失败类型 | 触发条件 | 结果 |
|----------|----------|------|
| 普通失败 | 默认 | 进度-30%，无伤害 |
| 重伤失败 | 连续2次失败 | 进度-50%，寿元-5年 |
| 走火入魔 | d100 ≤ 5 | 进度清零，随机属性永久-2 |
| 身死道消 | 元婴以上 + d100 ≤ 1 | 弟子死亡 |
| 天劫失败 | 金丹以上大突破 | 修为倒退一个境界（大惩罚） |

## 三、灵根系统

### 五行属性
```
金 木 水 火 土
```

### 灵根品质判定

| 品质 | 属性数 | 稀有度 | 修炼倍率 | 突破倍率 |
|------|--------|--------|----------|----------|
| 天灵根 | 1种 | 0.5% | 2.0x | 2.0x |
| 异灵根 | 2种（相生）| 5% | 1.5x | 1.5x |
| 真灵根 | 3种 | 20% | 1.0x | 1.0x |
| 伪灵根 | 4种 | 45% | 0.6x | 0.6x |
| 废灵根 | 5种 | 29.5% | 0.3x | 0.3x |

### 五行相生相克

**相生**: 金→水→木→火→土→金（相生属性越多，灵根品质越好）
**相克**: 金克木，木克土，土克水，水克火，火克金

相克的属性组合如果出现在灵根中，品质自动降一级。

### 修炼速度公式

```
每日修为获取 = 基础值 × 灵根倍率 × 设施加成 × 丹药效果 × 灵气浓度
```

| 参数 | 值 |
|------|-----|
| 基础值 | 1.0 |
| 修炼室Lv.1 | +20% |
| 修炼室Lv.2 | +40% |
| 修炼室Lv.3 | +60% |
| 下品灵脉 | +10% |
| 中品灵脉 | +25% |
| 上品灵脉 | +50% |
| 极品灵脉 | +100% |
| 聚灵丹 | +30%（持续3个月） |

### 药老体质（隐藏机制）

极小概率（0.01%）出现"药老体质"的弟子 — 废灵根但拥有隐藏修炼潜力，在突破元婴后灵根自动晋升为天灵根。

## 四、天劫系统

### 天劫类型

| 天劫 | 触发条件 | 雷劫数 | 基础存活率 |
|------|----------|--------|-----------|
| 小天劫 | 结金丹突破 | 3道 | 80% |
| 四九天劫 | 结元婴突破 | 9道 | 60% |
| 六九天劫 | 化神突破 | 18道 | 40% |
| 心魔劫 | 炼虚突破 | 非雷劫 | 取决于心性 |
| 九九重劫 | 合体突破 | 27道 | 20% |
| 飞升劫 | 大乘突破 | 81道 | 5% |

### 天劫存活公式

```
雷劫存活率 = 基础存活率
    × (1 + 根骨/100)
    × (1 + 心性/200)
    × (1 + 渡劫阵法加成)
    × (1 + 渡劫丹加成)

实际结果 = d100 < (存活率 × 100) ? 成功 : 失败
```

### 渡劫辅助
| 辅助手段 | 加成 |
|----------|------|
| 渡劫丹 | +15% |
| 护山大阵 Lv.3 | +10% |
| 长老护法 | +5%/每位元婴以上长老 |
| 法宝（防御类） | +5-20%因品阶而定 |
