# 修仙：宗门风云完整游戏方案

## 一句话定位

玩家扮演一位新任掌门，从破败山门起步，通过收徒、建设、任职、处理宗门纪事、观察天下局势，把宗门经营成修真界大宗。

## 设计支柱

1. **掌门视角**：玩家管理宗门，而不是扮演单个修士。
2. **弟子是核心资产**：每名弟子都有擅长、性格、来历、记忆和职位适性。
3. **宗门可见成长**：建设和升级要改变宗门地图，而不只是增加数值。
4. **故事藏在经营里**：主线不强推，通过纪事、地图异动、弟子经历和天下榜变化逐步显露。
5. **局势压力清楚**：玩家能看见本门和其他宗门的差距，并知道短板在哪里。

## 核心循环

### 月度循环

1. 查看掌门总览。
2. 处理宗门纪事。
3. 调整弟子任务和职位。
4. 建设或升级设施。
5. 推进一月并查看反馈。

### 年度循环

1. 查看天下榜和宗门差距。
2. 判断宗门发展方向：修炼、经济、炼丹、战力、外交。
3. 处理世界局势变化：妖兽潮、魔修异动、宗门大会、秘境开放。

### 长期循环

1. 培养核心弟子。
2. 建立堂口与职责分工。
3. 推动本门旧阵基、人魔大战、宗门倒戈等暗线。
4. 从九品宗门晋升到一品宗门。

## 前期体验

前 12 个月用「掌门初任目标」引导玩家：

- 招收一名新弟子。
- 建设或升级一座设施。
- 任命一名副长老或护法。
- 处理一件宗门纪事。

四个目标完成后，解锁「掌门初任」故事线索，让玩家理解：经营行为本身会推动故事发现。

## 系统规划

### 掌门总览

掌门总览是默认首页，承担经营游戏的信息中枢：

- 本月收入/支出。
- 弟子状态分布。
- 宗门排名与差距。
- 待处理纪事。
- 掌门初任目标或当前阶段目标。
- 推荐行动。

### 弟子系统

弟子资料应包含：

- 基础属性：年龄、境界、灵根、寿元。
- 擅长方向：剑修、丹道、阵法、灵植、驭兽、符箓等。
- 性格：勇猛、谨慎、善良、孤傲、阴狠等。
- 来历故事：入门前经历。
- 宗门履历：入门、突破、任职、事件处理、战斗经历。
- 心境风险：忠诚、压力、魔念、野心。

### 宗门建设

宗门地图是美术和经营成就感核心：

- 每座设施在宗门地图上有位置。
- 建设/升级会改变建筑外观。
- 弟子会显示在对应地点。
- 宗门整体阶段从破败山门、初兴宗门、名门大派逐步变化。

### 宗门纪事

纪事是消息中心，不再弹窗打断：

- 待处理。
- 影响中。
- 已归档。
- 天下传闻。
- 弟子私事。

处理流程：

1. 选择委派弟子。
2. 查看该弟子的性格判断。
3. 选择方案。
4. 确认处理。
5. 记录影响和记忆。

### 堂口系统

中期开始引入堂口：

- 丹堂：炼丹、药材、丹师培养。
- 战堂：防御、妖兽、宗门战。
- 外务堂：外交、商队、情报。
- 传功堂：教学、突破辅导。
- 执法堂：纪律、叛逃、门规。

堂口负责人能力和性格会影响自动处理质量。

### 天下局势

天下榜和世界地图提供长期压力：

- 综合榜。
- 战力榜。
- 财富榜。
- 声望榜。
- 炼丹榜。
- 正魔倾向。
- 宗门关系。
- 地图异动。

### 大事件

大事件需要预兆，不应突然砸脸：

- 弟子叛逃：忠诚下降、私藏资源、同门冲突。
- 魔修入侵：魔气增强、商队失踪、护山阵报警。
- 宗门大会：邀请函、报名期、备战期、比武期。
- 灵脉争夺：地图标记、外交施压、战斗或谈判。

## 美术方向

风格关键词：**水墨卷轴、工笔人物、深色经营 UI、金玉点缀、宗门可见成长**。

优先资产：

1. 宗门地图阶段图。
2. 设施升级图标或建筑小图。
3. 弟子头像池。
4. 事件插画。
5. 世界地图标记。
6. UI 组件：按钮、面板、红点、榜单条、进度条。

AI 美术流程：

1. 建立 Art Bible。
2. 固定 prompt 模板。
3. 生成多版本。
4. 人工筛选。
5. 统一裁切、压缩、命名。
6. Godot 统一导入规格。

## 技术路线

继续使用 Godot。当前问题不是引擎能力，而是玩法结构、UI 架构和资产规范。

需要改进：

- 固定坐标 UI 逐步改为 Container 响应式布局。
- 事件、目标、弟子故事、堂口职责数据化。
- 平衡数据逐步迁移到表格或独立配置。
- 建立烟测和回归测试，覆盖核心经营循环。

## 开发阶段

### 阶段 1：经营骨架

- 掌门总览。
- 掌门初任目标。
- 统一导航。
- 纪事消息中心。

### 阶段 2：弟子深化

- 弟子卡片列表。
- 性格影响更多系统。
- 个人故事线。
- 任职和堂口职责。

### 阶段 3：宗门可视化

- 宗门地图设施状态。
- 弟子位置。
- 建设/升级外观变化。
- 宗门阶段变化。

### 阶段 4：大事件与天下局势

- 魔修入侵链。
- 弟子叛逃链。
- 宗门大会。
- 宗门倒戈。
- 世界榜单与地图联动。
