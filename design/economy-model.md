## 四、价格曲线

### 丹药基准价

| 丹药 | 基础价格（下品灵石） | 炼制成本 |
|------|---------------------|----------|
| 聚灵丹（下品） | 50 | 30 |
| 聚灵丹（中品） | 200 | 100 |
| 聚灵丹（上品） | 800 | 350 |
| 筑基丹 | 500 | 200 |
| 结金丹 | 3000 | 1200 |
| 结婴丹 | 15,000 | 6000 |
| 化神丹 | 80,000 | 30,000 |

### 价格波动机制

```
坊市价格 = 基础价格 × 供需系数 × 随机波动

供需系数:
- 正魔大战期间: ×1.5-2.0（丹药需求激增）
- 和平时期: ×0.8-1.0
- 当地无该丹药产出: ×1.3

随机波动: ×0.8-1.2
```

## 五、升级曲线

### 设施建造成本递进

```
每级成本 = 基础成本 × 3^(等级-1)

例 - 修炼室:
Lv.1: 100
Lv.2: 300
Lv.3: 900
```

### 灵石产出升级

```
每级产出 = 基础产出 × 2^(等级-1)

例 - 下品灵脉:
Lv.1: 10/月
Lv.2: 20/月
Lv.3: 40/月
Lv.4: 80/月
Lv.5: 160/月
```

### 弟子修为进度

```
练气: ~5年达到9层（天灵根）、~35年（废灵根）
筑基→金丹: ~10年（天灵根）、~70年（废灵根）
金丹→元婴: ~50年（天灵根，含丹药）
```

## 六、平衡目标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 首个弟子突破筑基 | 游戏内5-10年 | 天灵根+筑基丹 |
| 首次灵石月结余为正 | 第3-6个月 | 建好第一座灵脉后 |
| 建造第二座设施 | 第12-18个月 | 需要攒够灵石 |
| 第一个金丹弟子 | 游戏内20-40年 | 天灵根+结金丹+顺利渡劫 |
| 宗门升八品 | 游戏内30-50年 | 金丹弟子+资源积累 |
| F2P等价满级用时 | ~500年（游戏内） | 一品宗门+多名元婴 |

## 七、经济健康指标

### 监测指标

| 指标 | 健康范围 | 警告阈值 |
|------|----------|----------|
| 月灵石增长率 | 5-15% | <0% 或 >30% |
| 灵石产出/消耗比 | 1.1-1.5 | <0.9 或 >2.0 |
| 弟子人均灵石消耗 | 10-50/月 | 随境界合理增长 |
| 丹药通胀率（百年） | <20% | >50% |
| 最富宗门/最穷宗门比 | <10:1 | >20:1 |

### 自动平衡机制

- 如果灵石产出过高 → 触发"灵石矿脉枯竭"事件
- 如果灵石产出过低 → 触发"发现新灵脉"事件
- NPC宗门的经济也遵循相同模型

## 八、道德护栏

| 原则 | 实施 |
|------|------|
| **无付费赢** | 单机买断，无内购 |
| **透明概率** | 所有成功率在UI上明确显示 |
| **无赌徒机制** | 关键突破有保底（3次失败后第4次成功率翻倍） |
| **尊重玩家时间** | 可加速时间，无强制等待 |
